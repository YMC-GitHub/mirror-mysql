## 检查 docker 状态 | win
```powershell
# Check Docker version in WSL
docker --version

# Verify Docker is running
docker ps
```

##  检查 docker 上下文 | win
```powershell
# 列出所有可用的 Docker 上下文
docker context ls

# 查看当前使用的 Docker 上下文
docker context inspect 
```

## 检查 docker 状态 | wsl 
```bash
# Check Docker version in WSL
docker --version

# Verify Docker is running
docker ps

docker info
```
